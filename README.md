# ☕ Coffee Machine Program (Python)

A simple **console-based coffee machine simulation** built with Python.

Order coffee, insert coins, and the machine will prepare your drink if enough resources are available! ☕💰

---

## 📌 About the Project

<details>
<summary><strong>▶ What does this project do?</strong></summary>

This project simulates the working of a **coffee vending machine**.

The user can choose between different drinks, and the machine will:

- 🧾 Check if there are enough ingredients  
- 💰 Ask for coin payment  
- 💵 Return change if necessary  
- ☕ Prepare the selected drink  

It also allows viewing a **resource report** and turning the machine **off**.

This beginner-friendly project helps practice:

- 🧠 Functions  
- 📋 Dictionaries  
- 🔁 `while` loops  
- 🔀 Conditional statements  
- 💰 Basic transaction logic  
- 🐍 Structuring real-world programs  

</details>

---

## ✨ Features

- ☕ Multiple drink options (Espresso, Latte, Cappuccino)  
- 🧾 Ingredient resource tracking  
- 💰 Coin-based payment system  
- 💵 Automatic change calculation  
- 📊 Resource report command  
- 🔌 Turn off the machine
